Сколько существует способов задать переменную типа slice или map?

Для слайса (slice):

slice := make([]int, 5) // Создать слайс длиной 5
slice := []int{1, 2, 3, 4, 5} // Создать слайс и инициализировать его значениями
var slice []int // Объявить слайс с нулевым значением (nil)

existingSlice := []int{1, 2, 3}
newSlice := existingSlice[1:2] // Создать слайс, используя срез другого слайса

Для карты (map):

myMap := make(map[string]int) // Создать карту
myMap := map[string]int{"one": 1, "two": 2, "three": 3} // Создать карту и инициализировать её парами ключ-значение
var myMap map[string]int // Объявить карту с нулевым значением (nil)
