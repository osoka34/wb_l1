В чем разница make и new?

make: Эта функция используется для создания экземпляров слайсов, карт (maps) и каналов (channels),
которые являются ссылочными типами данных (reference types) в Go.
make инициализирует и выделяет память для этих структур данных и возвращает их готовые к использованию.

slice := make([]int, 5)       // Создать слайс с длиной 5
mymap := make(map[string]int)  // Создать карту
ch := make(chan int)           // Создать канал

new: Эта функция используется для выделения памяти для новых экземпляров значимых типов (value types),
таких как указатели, структуры и базовые типы данных (int, float, bool и т. д.).
new возвращает указатель на выделенный объект,
а объекты инициализируются нулевыми значениями по умолчанию.

var myInt *int
myInt = new(int)  // Создать указатель на int и выделить для него память

type Person struct {
    Name string
    Age  int
}

p := new(Person)  // Создать указатель на структуру Person
